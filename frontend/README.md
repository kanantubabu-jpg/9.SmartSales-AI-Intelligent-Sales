# Frontend (React + Vite)

This frontend is built with Vite and React. It connects to the backend API and provides the main dashboard UI.

## Quick start

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the production bundle:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Notes

- The frontend uses `vite` for fast HMR during development.
- Production assets are generated in `frontend/dist`.
- Use `npm run build` before deploying the frontend.
